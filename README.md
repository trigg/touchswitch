# touchswitch

A touch orientated switcher for wayfire



https://github.com/user-attachments/assets/e742e858-d5f3-4246-b4f8-a03d6fa684e1



## Installing

Set `--prefix` to the same as the wayfire installation.

```
$ meson setup build --prefix=/usr
$ ninja -C build
# ninja -C build install
```

Restart wayfire.
